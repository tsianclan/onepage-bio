Interests

  • Software Defined Networking - A network architecture that decouples the control plane from the forwarding plane leading to a programmable abstracted infrastructure.
  • Software Defined - WAN - An SDN use case where overlay tunnels are used to allow the WAN become transport agnostic and optimise application performance over lower cost access circuits.
  • Network Function Virtualisation - A network architecture that decouples the networking functions from the hardware allowing them to be run as software over commodity servers, offering flexibility and reducing CAPEX & OPEX.
  • Programmable Data Plane - A new networking paradigm that abstracts the forwarding plane and allows switches to be controlled by software in order to perform complex and versatile operations.
  • Networking as a Service - A principal where users perceive a multi-provider network as an abstract entity offering end-to-end value-added services.
